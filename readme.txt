Implemented Features:
- Complete Adrenaline Rules
- Command Line Interface
- Graphic User Interface (JavaFX)
- Socket Connections
- RMI Connections
- Multiple Games (Advanced Feature)

How to run the software:

# Server
Run "java -jar /deliveries/final/JAR/Adrenaline-1.0-server.jar" with Bash/CMD

OR

Double click/call /deliveries/final/JAR/server.sh  (on Unix-like)
Double click/call /deliveries/final/JAR/server.bat (on Windows)

# Client (CLI)
Run "java -jar /deliveries/final/JAR/Adrenaline-1.0-CLI.jar" with Bash/CMD

OR

Double click/call /deliveries/final/JAR/client-cli.sh  (on Unix-like)
Double click/call /deliveries/final/JAR/client-cli.bat (on Windows)

# Client (GUI)
Run "java -jar /deliveries/final/JAR/Adrenaline-1.0-GUI.jar" with Bash/CMD

OR

Double click/call /deliveries/final/JAR/client-gui.sh  (on Unix-like)
Double click/call /deliveries/final/JAR/client-gui.bat (on Windows)


IMPORTANT NOTE: make sure to have the libs/ folder available and visible to all the jar files.
